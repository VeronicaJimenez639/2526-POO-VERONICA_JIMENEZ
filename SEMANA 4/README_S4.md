# Ejemplo del Mundo Real - Biblioteca (POO)

## Caso del mundo real
Este programa simula una biblioteca sencilla donde existen libros identificados por ISBN.
Un usuario puede prestar un libro si está disponible y también puede devolverlo.
La biblioteca busca libros por ISBN para prestar o devolver.

## Clases usadas
- Libro: tiene título, autor, isbn y estado (prestado/disponible)
- Usuario: tiene nombre y cédula
- Biblioteca: guarda una lista de libros y gestiona préstamos/devoluciones

## Cómo ejecutar
1) Abrir la carpeta biblioteca_app en PyCharm
2) Ejecutar:
   python main.py
